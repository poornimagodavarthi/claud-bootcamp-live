# Claude Code 101 — Beginner Slide Decks

| #  | Slug                                                                     | Duration | Description                                                            |

| 02 | [part-02-first-conversation.md](part-02-first-conversation.md)           | 25 min   | Your first real conversation: multi-turn, accept, reject.              |
| 03 | [part-03-asking-for-what-you-want.md](part-03-asking-for-what-you-want.md) | 30 min | Sharp prompts: role + goal + constraint + format.                      |
| 04 | [part-04-reading-code-together.md](part-04-reading-code-together.md)     | 25 min   | Explain unfamiliar code in plain English.                              |
| 05 | [part-05-editing-one-file-safely.md](part-05-editing-one-file-safely.md) | 30 min   | First real edit, with Git as the safety net.                           |
| 06 | [part-06-claude-md-cheat-sheet.md](part-06-claude-md-cheat-sheet.md)     | 25 min   | A 15-line CLAUDE.md that Claude reads every session.                   |
| 07 | [part-07-safer-and-smarter.md](part-07-safer-and-smarter.md)                     | 25 min   | What never to paste: secrets, PII, proprietary data.                   |
