# Module 8 — Reference Material

The `before/` folder contains a deliberately messy `pricing.py` and a passing test suite. Students copy it to `module-08/after/` and refactor under hard constraints.

## Run the tests

```bash
cd before/
pip install pytest
pytest -q
```

All tests must remain green after the refactor — that's the contract.

## Reference HANDOFF and ARCHITECTURE

A reference `HANDOFF.md` and `ARCHITECTURE.md` live below; instructors compare student output against them but accept any version that satisfies the deliverable checklist in the exercise README.
