# How to use this book

*Claude Code Bootcamp — Build 10 Real-World Projects with Claude Code*  
Instructor: Luca Berton · Edition: May 2026

This book is the complete Claude Code Bootcamp. It compiles the workshop's eleven parts into continuous prose, with the hands-on exercises embedded in each chapter so you can build as you read. You direct Claude Code; it implements; you review and merge. You are always the engineer of record.

Every exercise and reference solution also lives in the companion repository, the [Claude Code Bootcamp repository](https://github.com/lucab85/Claude-Code-Bootcamp). Clone it and work each module from the live files; each chapter links directly to its exercise there.

Reference solutions are included at the end of each project chapter. Treat them as a checklist to compare against *after* you have produced your own deliverable — not as something to copy.

# Prerequisites and pre-work

Complete this checklist before working through the chapters (~30 min):

- Claude Code installed and signed in (any tier)
- Python 3.11+ on `PATH` (primary track)
- Node.js 20+ on `PATH` (secondary track for parts 2/4/5)
- Git >= 2.30 on `PATH`
- An IDE you can drive (VS Code recommended)
- macOS / Linux / Windows-via-WSL2 — native PowerShell is not supported
