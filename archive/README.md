# Archive

This directory holds **optional pre-bootcamp warm-up material** that is NOT part of the 4-hour bootcamp agenda.

It is kept for historical reference and for students who want a gentler on-ramp before the main 10-part programme.

- `beginner/` — the earlier beginner-track slides, exercises, assessments, and guides (formerly `slides/beginner/`, `exercises/beginner/`, `assessments/beginner/`, and the root `beginner-*.md` files).

**Policy**

- No bootcamp module (parts 01–10 or the Part 11 closing block) links into this directory.
- The only entry point is a single labelled section in the root `README.md`: **Optional pre-bootcamp warm-up (archived)**.
- The audit gate `audit.archive-isolation` enforces this rule.

If you are running the 4-hour bootcamp, **do not point students here** during the workshop. Mention it only at the end as a self-study option.
